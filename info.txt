// PROCESO PARA GENERAR EL ENTORNO VIRTUAL + REPOSITORIO EN GIT + HOJA DE REQUERIMIENTOS (ejemplo realizado con PANDAS:

1. Generación de entorno virtual en la carpeta del proyecto
 - Python3 -m venv  blog_env

1.1 He activado el entorno virtual y he descargado la librería pandas (pip install pandas)
1.2 He desactivado el entorno virtual
- deactivate

2. He generado un archivo local de git
Git inita

2.1 He aplicado el siguiente comando:
- echo 'env' > .gitignore (para poder ignorar de las subidas las carpetas del entorno virtual)

2.2 He aplicado el siguiente comando: 
- pip freeze > requirements.txt (para poder tener en un archivo de texto todas las dependencias)

2.3 He creado el repo en GitHub
- he copiado el path de la carpeta

2.4 En la consola en el path de interés he aplicado el siguiente código: 
- git remote add origin https://github.com/ManuData/BLOG.git

2.5 He realizado commit: 
git commit -m "subida requerimientos de pandas"

2.6 He generado el push:

 git push -u origin master
